TITLE:
🚀 Enhancement: Add  `Query.deselect` to Queries

ISSUE BODY:
### 🔖 Enhancement description

`Query.select` enables us to select the fields we need but how about we get a lot of fields and we want to exclude a field? There is no handy way to do so.

### 🎤 Pitch

`Query.deselect` should do the opposite of `Query.select` do.

> Query.deselect(["name", "title"])
